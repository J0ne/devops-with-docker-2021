### 1.13: Hello, backend!

Dockerfile:
```sh
FROM golang:1.16
WORKDIR /usr/src/app
COPY /example-backend /usr/src/app
RUN go build
ENV REQUEST_ORIGIN=http://localhost:5000
ENV PORT=8080
ENTRYPOINT [ "./server" ]
```
Command: 
```sh
docker build . -t back && docker run -p 8080:8080 back
```
